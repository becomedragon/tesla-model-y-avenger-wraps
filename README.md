## How to Use

### 1. Download the Images
Download `iron_man_wrap.png` and `captain_america_wrap.png` from this repository.

### 2. Prepare Your USB Drive
- Format your USB drive as exFAT, FAT32, or ext3/ext4 (NTFS is NOT supported)
- Create a folder named `Wraps` at the root level of the drive
- Place the PNG files inside the `Wraps` folder
- Ensure the drive doesn't contain map or firmware updates

### 3. Load into Your Tesla
1. Insert the USB drive into your Model Y
2. Go to: **Toybox ‚Üí Paint Shop ‚Üí Wraps tab**
3. Select and apply your chosen wrap
